# LaTeX code Viewer by Saptarshi Dey

## A very basic LaTeX code previewer built using MathJax and VsCode WebView extension
## Getting started
- **Step-1:** <kbd>Ctrl+Shift+P</kbd>
- **Step-2:**

      latex-viewer.start
